# Qwen3-8B jax v004 — gradient remat + bs=2

## Hypothesis under test

**Hypothesis**: Per-layer `jax.checkpoint` (nothing_saveable) makes bs=2 (global 16)
fit at seq2048 — the shape that OOM'd in [v001](2026-06-02-v001-batch-bs2.md) at
43.12 G — and the added occupancy beats the bs=1 baseline (20.5% MFU).

**Mechanism**: `--use_remat` wraps each Qwen3DecoderLayer in `jax.checkpoint`, so
only one layer's activations are live during backward recompute. Image
`qwen3-8b-jax:v004-remat`. bs=2/seq2048.

**Predicted signal**: compile HBM < 31.25 G (no OOM); MFU ≥ 20.5%.

**Falsification criterion**: still OOMs, or MFU < 20.5%.

## Setup

- v6e-8 (fsdp=8, tp=1), image `qwen3-8b-jax:v004-remat`, `--use_remat=True --batch_size=2 --seqlen=2048`.
- global_batch 16, per-chip 2, bf16, 20 steps. `[remat] per-layer jax.checkpoint … ON` confirmed in log.

## Baseline comparison

| Metric | Baseline (bs=1) | **v004 (remat bs=2)** | Δ |
|--------|-----------------|------------------------|---|
| Fits at seq2048 | yes | **yes** (bs=2 OOM'd without remat) | unlock |
| Throughput | 3,994 tok/s/chip | **4,280** | **+7.3%** |
| MFU (effective) | 20.5% | **22.0%** | **+1.5 pp** |
| MXU util (xprof) | 19.9% | **27.8%** | +7.9 pp |

## Results

| Metric | Value |
|--------|-------|
| Steady step time | ~957 ms (global 16) |
| Throughput | **34,243 tok/s (4,280/chip)**, 18 steps measured |
| MFU | **22.0%** (trainer) / MXU util 27.8% (xprof) |
| Loss | 12.09 → 12.08 (flat, synthetic — semantics OK) |
| Exit | 0 |

Remat adds ~1.3× recompute, but the bs=2 **occupancy gain more than pays for it**:
effective MFU rises 20.5% → 22.0%, and hardware MXU util jumps to 27.8%.

## Verdict

**supported — NEW FRONTIER.** Gradient remat unlocks bs=2 (which OOM'd without it)
and lifts effective MFU to **22.0%** (4,280 tok/s/chip), +1.5 pp / +7.3% over the
bs=1 baseline. The remat code change is committed to the lane trainer and becomes
the **stable image** (`qwen3-8b-jax:v004-remat`); remat is now the default for
subsequent experiments. Next: push batch further (remat+bs=3), and splash (now
wired) to remove the N² scores and reach seq8192 / bs=4.
